========================================================================
    ToyRobot Project Overview
========================================================================

Build the source code:
Use Microsoft Visual Studio 2012 or later, to open the solution file.

Run the program:
Once it's been built, execute the program with syntax as follows:
ToyRobot <input file>

for instance if a text file named test.txt(as in the project folder) containing input commands are placed in the same path
as the executable file, just type: 
ToyRobot test.txt

Usable commands are those listed in test instructions:
PLACE, MOVE, LEFT, RIGHT, REPORT
